How It Works

Get started in minutes with a simple setup process

1

Configure Your Templates & Settings

Set up templates with output paths, file naming conventions, and processing types (Simple, Regex, JavaScript, or AI-Powered). Configure ProPresenter connections, AI providers, network sync, and other advanced settings.

2

Link Files in ProPresenter

Use ProPresenter's Linked Text feature to connect text boxes to the files ProAssist generates. Each slide line links to a separate file. Configure presentation activation for automatic slide changes.

3

Import & Process Content

Import from text files, Live Slides sessions, network masters, or use AI to generate content. ProAssist automatically splits and formats content according to your template's rules or AI instructions. Use the Global AI Assistant for natural language processing.

4

Go Live & Automate

Click any slide to instantly update ProPresenter. Use Stage Assist for timer management and schedule automation. Leverage SmartVerses for Bible verses, Live Testimonies for speaker management, and Live Slides for real-time collaboration.
