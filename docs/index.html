How It Works

Get started in minutes with a simple setup process

1

Configure Your Templates & Settings

Set up templates with output paths, file naming conventions, and processing types (Simple, Regex, JavaScript, or AI-Powered). Configure ProPresenter connections, AI providers, network sync, and other advanced settings.

2

Link Files in ProPresenter

Use ProPresenter's Linked Text feature to connect text boxes to the files SmartVerses generates. Each slide line links to a separate file. Configure presentation activation for automatic slide changes.

3

Import & Process Content

Import from text files, Live Slides sessions, network masters, or use AI to generate content. SmartVerses automatically splits and formats content according to your template's rules or AI instructions. Use the Global AI Assistant for natural language processing.

4

Go Live & Automate

Click any slide to instantly update ProPresenter. Use Smart Timers for AI-powered timer management and schedule automation. Leverage SmartVerses for Bible verses, Live Testimonies for speaker management, and Live Slides for real-time collaboration.

1

Set Up Your AI & Models

Configure your AI processes and download the models you need—choose offline, cloud-based, or completely free options. SmartVerses works with a range of providers so you can match your setup to your budget and privacy requirements.

2

Choose Your Audience Display

Select which screen to use for your audience display, then customize the look: pick your background, font, and layout. Everything is configurable so your output fits your service style.

3

Go Live

Start your session and run SmartVerses on its own—no ProPresenter required. Live transcription, verse detection, key points, and audience display all work together so you can focus on your message.

More Capabilities

Powerful tools for professional production teams

Flexible Templates

Four processing types: Simple, Regex, JavaScript, and AI-Powered. Create custom templates with color coding, icons, and layout options for any content type.

Real-time Collaboration

WebSocket-powered Live Slides let your team edit from any browser. Share URLs, sync across devices, and see changes instantly. Perfect for remote teams and distributed workflows.

Live Slides Network Sync Remote Access

Live Transcription

Real-time speech-to-text with AssemblyAI integration. Automatically detect Bible references in speech and trigger ProPresenter slides instantly.

Smart Bible Search

Direct reference parsing (John 3:16) or AI-powered phrase search. Navigate between verses, detect paraphrased content, and extract key points.

Schedule Automation

Automatically trigger ProPresenter slides and change stage layouts when timer sessions start. Dynamic time adjustment keeps everything on track.

Remote Access

Access timer controls remotely via shareable links. Perfect for stage managers and production teams working from different locations.

Multi-Device Sync

Keep playlists and schedules synchronized across multiple devices. Master/Slave/Peer modes ensure everyone stays in sync.

ProPresenter Control

Control multiple ProPresenter instances simultaneously. Configure per-slide activation, stage layouts, timer sync, and custom click patterns.

macOS Installation Note

Important: This is an open-source project and does not have an Apple Developer account signing the app. To run on macOS:

Option 1 (Recommended)

Right-click the DMG file → Open (instead of double-clicking)

Option 2

Run in Terminal:

xattr -cr /path/to/proassist.app

Replace /path/to/proassist.app with the actual path to the application after mounting the DMG.
