Built for ProPresenter Users

AI-Powered Automation for ProPresenter

ProAssist helps ProPresenter users spend less time preparing slides and more time focusing on the service. It uses AI to process and format text into ready to use slides, removing repetitive copy and paste tasks. With added automation tools like timers and command based control, ProAssist simplifies both preparation and live operation. Leveraging LLMs, it also allows users to chat with the system and give natural language instructions to make changes directly in ProPresenter.

How It Works

Get started in minutes with a simple setup process

1

Configure Your Templates

Set up templates with your preferred output paths, file naming conventions, and processing types. ProAssist writes to text files that ProPresenter reads.

2

Link Files in ProPresenter

Use ProPresenter's Linked Text feature to connect text boxes to the files ProAssist generates. Each slide line links to a separate file.

3

Import & Process Content

Paste your text or upload files. ProAssist automatically splits and formats content according to your template's rules or AI instructions.

4

Go Live!

Click any slide to instantly update ProPresenter. Your content appears on screen immediately, perfectly formatted every time.
